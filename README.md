# Ai-Resume-Analyzer
Built a Built an AI powered resume analyzer that helps users compare their resumes with job descriptions and understand how well they match. The tool analyzes skills and keywords, highlights missing requirements, and provides suggestions to improve resume alignment with specific roles. Added resume upload support and designed a clean, user-friendly interface to make the analysis process simple and accessible.


-> What it does:
Upload your resume (PDF / image / or paste text)
Paste a job description
Get a match score (0–100%)
See which skills you already have
See what skills are missing
Get basic improvement tips
